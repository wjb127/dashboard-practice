## Soft Pastel Dashboard Style Module

> 부드럽고 따뜻한 파스텔 톤의 대시보드

**적합 산업**: 뷰티, 화장품, 베이커리, 카페, 육아, 교육, 웰니스, 요가

**핵심 특징**:
- 파스텔 핑크/보라/블루 색상
- 둥근 모서리 (rounded-3xl)
- 그래디언트 배경
- 부드러운 그림자

**주요 컴포넌트**:
- `SoftPastelCard` - 파스텔 그래디언트 카드
- `SoftPastelButton` - 둥근 버튼
- `SoftPastelKPICard` - 귀여운 KPI 카드

**색상 팔레트**: Pink #f472b6, Purple #c4b5fd, Blue #38bdf8, Peach #fb923c, Mint #4ade80
